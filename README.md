# MoneyManager - 家計管理アプリ

シンプルで使いやすい家計管理Webアプリです。ブラウザだけで動作し、データはlocalStorageに保存されます。

## 公開サイト

**[→ アプリを開く](https://takumayellow.github.io/moneymanagement/)**

## 機能

- **収支の記録**: 支出・収入をカテゴリ付きで記録
- **カテゴリ選択**: 食費 / 交通費 / 娯楽 / 光熱費 / 住居費 / 医療費 / 衣服費 / 通信費 / 給与 / ボーナス など
- **サマリー表示**: 今月の収入・支出・残高をリアルタイム表示
- **カテゴリ別内訳**: 支出をカテゴリ別に集計してバーグラフで表示
- **月別グラフ**: Chart.js による月別収支グラフ（直近6か月）
- **日付順・金額順ソート**: 取引履歴を自由に並び替え
- **月別フィルター**: 特定月の取引だけ表示
- **削除確認ダイアログ**: 誤削除防止のconfirm表示
- **データ永続化**: localStorage で自動保存
- **レスポンシブデザイン**: スマートフォン・タブレット対応

## 使い方

`index.html` をブラウザで直接開くか、任意のWebサーバーで配信してください。

## 技術スタック

- HTML5 / CSS3 / Vanilla JavaScript (ES6+)
- [Chart.js 4](https://www.chartjs.org/) - グラフ描画
- [Google Fonts](https://fonts.google.com/) - Inter / Noto Sans JP
- localStorage - データ永続化

## データについて

全データはブラウザのlocalStorageに保存されます。ブラウザのデータを消去すると取引履歴も削除されます。
